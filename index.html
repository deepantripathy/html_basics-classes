Basic Syntax Of HTML

Now lets dig deep into the syntax. Here the expression <!DOCTYPE HTML> tells the version of the markup langauge you are using.

Then I used the <html> tag, now this is where the code lies both your head and body. Note :: the </html>, well this is basically the syntax, whatever you start it should end by a '/' like <a href=""> </a>.

The term head basically has the title of the webpage and all the links like .css and .js files. And body has the code of your website.


Body

In this section I will tell you about some commonly used tags.

  • <p> :: This tag is used for new paragraph
  • <a> :: This is basically the anchor tag used for providing links to other folders, websites etc.
  • <img> :: This is basically the image tag which is used for putting up some images.
  • <hi> :: This is the header tag. Here the letter i in hi can be {1,2,3,4,5,6}. h1 is the largest and size goes on decreasing.
  • <center> :: This tag is used for positioning a text exactly at the center.
  • And the list continues......

<a> tag

This is basically the anchor tag.

Eg. <a href="https://www.google.com"> go to google </a>.

Here if you click on "go to google" then you will be directed to the google website. This is infact useful for travelling between folders(not to worry this is my next section).


Travelling between Folders

Well travelling between folder means like you are on page 1 and you want to go to page 2 by just a click.

For eg.
<a href="test/test1/test2/data.html"> go to data.html</a>

As you can see this is the layout of "test/test1/test2/data.html" ie it enters test folder then test1 then test2 and then opens data.html .

This is basically going into further folders. Now if you want to come out of some folder, syntax is
<a href="../../../index.html"> go to index.html</a>

As you can see in this layout of "../../../index.html" ie the 1st '../' refers going from data.html to test2 folder, then 2nd '../' refers going from test2 folder to test1 folder, the 3rd '../' refers to going from test1 folder to test folder

This is all about travelling between folders.
